# Benefits Advisor

A specialist worker agent for benefits guidance at Zava Insurance. Connected to the HR Onboarding Buddy as a `worker_agent` for transparent delegation.

## Capabilities

| Capability | Source |
|------------|--------|
| 🧠 Instructions & persona | `appPackage/instruction.txt` |
| 📦 Embedded Knowledge | Benefits Guide |
| 📚 SharePoint | Benefits Documents (full plan docs, SBC documents, enrollment forms) |
| 🧮 Code Interpreter | Cost modeling, take-home pay calculations, plan comparisons |

## Scope

- Health insurance plan comparisons (Bronze, Silver, Gold)
- 401(k) matching and contribution modeling
- Parental leave and family benefits
- Wellness programs and stipends
- Enrollment workflows and deadlines
- Life event scenarios (marriage, new child, coverage changes)
- Take-home pay impact calculations

## Deployment

1. Open this folder in VS Code
2. Sign in with your Microsoft 365 account
3. Press **F5** to provision and deploy
4. Note the **M365 Title ID** from the output — the HR Onboarding Buddy needs it
