# IT Support Agent

A specialist worker agent for deep IT troubleshooting at Zava Insurance. Connected to the HR Onboarding Buddy as a `worker_agent` for transparent delegation.

## Capabilities

| Capability | Source |
|------------|--------|
| 🧠 Instructions & persona | `appPackage/instruction.txt` |
| 📦 Embedded Knowledge | IT Setup Guide |
| 📚 SharePoint | IT Knowledge Base (troubleshooting articles, known issues, device specs) |
| 🧮 Code Interpreter | Technical diagnostics |

## Scope

- Device troubleshooting (laptops, monitors, docks, peripherals)
- Software and access requests
- Network and connectivity (VPN, Wi-Fi, remote access)
- Account management (passwords, MFA, Entra ID)
- Email and collaboration tools
- Security incident guidance

## Deployment

1. Open this folder in VS Code
2. Sign in with your Microsoft 365 account
3. Press **F5** to provision and deploy
4. Note the **M365 Title ID** from the output — the HR Onboarding Buddy needs it
